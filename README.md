# Personal Portfolio Website

A simple personal portfolio website designed to showcase my skills, projects, and contact information.  
The site provides an overview of my technical background and allows visitors to explore the projects I have worked on.

## Live Demo

Visit the portfolio here:  
https://personal-portfolio-rho-three-33.vercel.app/

## Features

### Home
The landing section of the website that introduces the portfolio and serves as the entry point for visitors.

### About
Provides a brief overview of my background, experience, and interests in technology.

### Skills
Displays the various technologies and tools I have worked with.

### Projects
Showcases the projects I have built.  
Projects are displayed using a **button-based horizontal slider**, with links to their deployed versions.

### Contact
Allows visitors to send messages directly via email.  
This feature is implemented using **EmailJS**.

### Resume Download
The website includes a downloadable link for my resume.

## Technologies Used

- **HTML** – Website structure
- **CSS** – Styling and layout
- **JavaScript** – Interactivity and dynamic behavior
- **EmailJS** – Handling contact form email functionality

## Deployment

The website is deployed using **Vercel**.

## Running the Project Locally

To run this project on your local machine, follow the steps below:

### 1. Clone the Repository

Open a terminal (Command Prompt, PowerShell, Git Bash, or the terminal in VS Code) and run:

```bash
git clone https://github.com/YaBoiAce007/Personal-Portfolio.git
```

### 2. Navigate to the Project Folder

```bash
cd Personal-Portfolio
```

### 3. Open the Website

Locate the `index.html` file inside the project folder.

You can run the project by:

- Double-clicking the **index.html** file, or
- Right-clicking → **Open with** → your preferred web browser

Examples of supported browsers:
- Google Chrome
- Mozilla Firefox
- Microsoft Edge

The portfolio website will open in your browser.
